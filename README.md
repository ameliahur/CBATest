# Swagger Pet Store Regression Suite

## Requirements
- Python 3.8 or above

## How to Run
1. Clone the repository:
   ```bash
   git clone <GitHub_Repo_URL>
   cd swagger-petstore-regression

2. Set up the environment:
   ```bash
    python -m venv venv
    source venv/bin/activate  
    pip install pytest requests jsonschema

3. Run the tests: 
   ```bash
   pytest
